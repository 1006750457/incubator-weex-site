# <indicator>

# 简介

indicator 只能在 slider 中使用。

# 子组件

不可添加任何子元素。

# 属性

无扩展属性

# 样式

  • item-color, color. 指示点默认颜色（未选中态）。
  • item-selected-color, color. 指示点选中颜色。
  • item-size, number. 指示点半径。

# 事件

无事件

# 示例

请参考 slider
