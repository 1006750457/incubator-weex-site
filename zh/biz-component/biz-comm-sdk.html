# iOS

# 环境要求：

  • iOS 8.0+
  • CocoaPods

# 引入步骤：

iOS 的 EMAS-Weex 组件 使用 Cocoapods 进行包管理，在 Podfile 中加入如下代码即可引入

文件首部引入源：

source 'git@gitlab-ce.emas-poc.com:EMAS-Weex-iOS/weex-common.git'
source 'git@gitlab-ce.emas-poc.com:EMAS-Weex-iOS/native-common.git'

Pod 引入:

pod 'EmasWeexComponents'

Pod install 以后代码初始化即完成了组件的全部引入。

注：如需要使用 XSocial 的默认实现，请引入 EmasSocial 组件

pod 'EmasSocial'

这两个组件都需要您进行初始化，初始化流程如下

在初始化 WXSDKEngine 后，请用下面代码注册商业化组件

#import <EmasWeexComponents/EmasWeexComponents.h>

[EmasWeexComponents setup];

如果你引入了 EmasSocial 组件，因为其底层引入的是友盟分享套件，您可以移步友盟官网查看初始化流程，在 URL Types和 plist配置完成以后，也可以使用我们的SDK做初始化，初始化代码示例如下：

#import <EmasSocial/XCOSocial.h>

// 初始化友盟
[[XCOSocial sharedInstance] initUMWithUMAppKey:@"um_app_key" appSecrete:nil];

// 初始化 QQ
[[XCOSocial sharedInstance] setPlaform:UMSocialPlatformType_QQ appKey:@"app_key" appSecret:@"app_secret" redirectURL:@"http://mobile.umeng.com/social"];

// 初始化 Weibo
[[XCOSocial sharedInstance] setPlaform:UMSocialPlatformType_Sina appKey:@"app_key" appSecret:@"app_secret" redirectURL:@"http://sns.whalecloud.com/sina2/callback"];

// 初始化 微信
[[XCOSocial sharedInstance] setPlaform:UMSocialPlatformType_WechatSession appKey:@"app_key" appSecret:@"app_secret" redirectURL:@"http://mobile.umeng.com/social"];

# Android

# 环境要求

AndroidStudio

# 引入步骤

# 1. 添加相关软件库

在您的项目根目录下的build.gradle文件下加入软件仓库：

allprojects {
    repositories {
        ......
        maven {
            url "http://nexus-ce.emas-poc.com/repository/maven-public/"
        }

        maven {
            url "http://nexus-ce.emas-poc.com/repository/weex-common"
            credentials {
                username = ""
                password = ""
            }
        }

        maven {
            url "http://nexus-ce.emas-poc.com/repository/native-common"
            credentials {
                username = ""
                password = ""
            }
        }
    }

# 2. 引入相关依赖

在需要使用组件的module的build.gradle文件中添加对应依赖：

dependencies {
    
    ......
    
    // EMAS Native组件库，weex组件依赖xbase库，所以需要引入
    compile ('com.alibaba.emas.xcomponent:xbase:0.0.1')

    // EMAS Weex组件库
    compile ('com.emas.weex:weex-libs:0.0.1')
    compile ('com.emas.weex:weex-base:0.0.1')

    // EMAS weex指纹认证
    compile ('com.emas.weex:fingerprint:0.0.1')

    // EMAS weex手势密码
    compile ('com.emas.weex:patternlock:0.0.1')

    // 友盟社交分享相关库，请到友盟官网下载umeng-share SDK
    xxx

    // 友盟社交Weex组件实现
    compile ('com.alibaba.emas.xcomponent:umeng-social:0.0.1')
    
    // 高德定位SDK
    compile 'com.amap.api:location:latest.integration'
}

# 3. 初始化

分享组件基于友盟分享SDK实现，使用相关功能时需要在App启动化时进行相关初始化：

public class MyApplication extends Application {

    @Override
    public void onCreate() {
        super.onCreate();
        this.initSocial();
        ......
    }


    private void initSocial() {
        // 注册友盟分享、授权功能
        SocialManager.getInstance().registerShareService(UmengShareServiceImpl.class);
        SocialManager.getInstance().registerAuthService(UmengAuthServiceImpl.class);
        
        // 初始化友盟分享SDK
        Map<String, String> extras = new HashMap<>();
        extras.put("appkey", Constants.UMENG_APPKEY);
        extras.put("secret", Constants.UMENG_SECRET);
        extras.put("channel", Constants.UMENG_CHANNEL);
        SocialManager.getInstance().getShareService().init(this, extras);

        // 配置相关各社交平台相关参数
        SocialManager.getInstance().getShareService().setQQ(Constants.QQ_APPID, Constants.QQ_APPKEY);
        SocialManager.getInstance().getShareService().setWeibo(Constants.WEIBO_APPKEY, Constants.WEIBO_APPSECRET, Constants.WEIBO_REDIRECT_UTL);
        SocialManager.getInstance().getShareService().setWeixin(Constants.WEIXIN_APPID, Constants.WEIXIN_APPSECRET);
    }
}

友盟分享SDK的集成及配置可参考：友盟分享工程配置

另外引入高德地图功能需要在高德开放平台申请相关Key，具体操作可参考：高德地图官方文档
