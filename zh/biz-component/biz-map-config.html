# 配置项

本文列举初步拟定的配置项列表

# enableDataDecoratorAnimation: Boolean

是否允许图表动画开启

# enableGestureSimultaneously: Boolean

是否允许手势透传

# enableZoomInGestureOnXAxisDirection: Boolean

是否允许X轴方向的放缩手势，双指拉动可触发放缩

# enableZoomInGestureOnYAxisDirection: Boolean

是否允许Y轴方向的放缩手势，双指拉动可触发放缩

# userInteractionDisabled: Boolean

是否关闭用户交互

# xAxises: Array

控制x轴样式，第i个x轴的配置项$xAxis_i$ 为
$xAxis(i) = xAxises[i];$

[注] 目前最大支持2个横坐标轴

# xAxises[i].hidden: Boolean

控制x轴是否隐藏

# xAxises[i].xCategories: Array

传递x轴要展示的字符串数组，从0开始一一对应。
当数据位玫瑰图、饼图时，所能显示的最大数据数受xCategories的个数限制。

# xAxises[i].minimalGapBetweenTexts: Number

文字间的最小缝隙大小

# xAxises[i].labelTextColor: Color

坐标轴文字颜色

# xAxises[i].displayLabel: Boolean

是否展示坐标轴文本

# xAxises[i].axisLineWidth: Number

x坐标轴线条宽度，当为0时不显示坐标轴线。

# xAxises[i].axisLineColor: Color

x坐标轴坐标轴颜色，默认为灰色。

# xAxises[i].labelBias: Number

坐标轴文字和坐标轴之间的偏移距离。

# yAxises: Array

控制y轴样式，第i个y轴的配置项$yAxis_i$ 为

$yAxis(i) = yAxises[i];$

[注] 目前最大支持2个纵坐标轴

# yAxises[i].hidden: Boolean

控制y轴是否隐藏

# yAxises[i].commonFormat: String

通用的format类型，接受如下两种
- 'shortenNumber'	自动保留小数，并且会通过增加数字单位化简
- 'percentage'		百分比

# yAxises[i].isForcedStartAtZero: Boolean

是否强制从0开始，如果设置，跨越0的区域不会显示。

# yAxises[i].decimalScale: Number

默认formatter的小数位数，仅当formatter不被设置时候生效。

# yAxises[i].labelTextColor: Color

y坐标轴文字的颜色。

# yAxises[i].labelFontSize: String

y坐标轴文字的字体

# yAxises[i].displayLabel: Boolean

y坐标轴是否渲染文字

# yAxises[i].axisLineWidth: Number

y坐标轴线条宽度，当为0时不显示

# yAxises[i].axisLineColor: Color

y坐标轴坐标轴颜色

# yAxises[i].labelBias: Number

坐标轴文字和坐标轴之间的偏移距离

# grid: Map

图表的背景线

# grid.hidden: Boolean

是否隐藏

# grid.xAxisIndex: Number

背景线所依赖的x轴，设置后背景线的纵线和x轴的锚点一致

# grid.yAxisIndex: Number

背景线所依赖的y轴，设置后背景线的横线和y轴的锚点一致

# grid.lineWidth: Number

背景线的粗细

# grid.lineColor: Color

背景线的颜色

# grid.enableHorizontalLine: Boolean

是否展示横线

# grid.enableVerticalLine: Boolean

是否展示纵线

# highlight: Map

图表的高亮线

# highlight.hidden: Boolean

高亮线是否隐藏

# highlight.lineColor: Color

高亮线的线条颜色

# highlight.lineWidth: Number

高亮线线条粗细

# highlight.jointColor: Color

高亮线小圆球颜色

# highlight.jointSize: Number

高亮选中时小圆球的大小

# highlight.isDashLine: Boolean

高亮线是否为虚线

# highlight.shadowSize: Number

阴影大小，如果为0则不显示任何阴影

# highlight.shadowColor: Color

阴影颜色

# highlight.shadowOpaque: Number

设置阴影不透明度∈[0, 1]

# tooltip: Map

高亮线上的浮窗

# tooltip.commonFormat: String

- percentage 百分比
- shortenNumber 缩略到亿、万

# tooltip.hidden: Boolean

是否隐藏

# tooltip.formatter

格式化浮窗文字

# tooltip.displayXAxisValue: Boolean

是否展示x轴的值

# tooltip.displayDatasetValues: Boolean

是否展示dataset的值

# tooltip.decimalScale

当纵轴为数值轴的时候，默认格式化所用的保留小数位数

# tooltip.backgroundColor: Color

浮窗的背景颜色

# tooltip.cornerRadius

浮窗上的圆角大小

# tooltip.borderColor: Color

浮窗边框颜色

# tooltip.borderWidth

浮窗边框线条粗细

# legends: Map

图例

# legends.hidden: Boolean

是否展示图例

# pieIndicator: Map

# pieIndicator.hidden: Boolean

针对饼图生效，是否隐藏每个数据的描述

# series: Array

数据序列

# series[i].type: String

用于表示当前数据的类型
	- 'bar'
	- 'stackbar'
	- 'scatter'
	- 'stackline'
	- 'line'
	- 'pie'
	- 'radar'
	- 'rose'

# series[i].displaySelectedLabel: Boolean

是否展示选中点的label

# series[i].displaySelectLabelTextColor: Color

选中点的label颜色

# series[i].displaySelectLabelTextFontSize: Font

选中点的label的字体大小

# series[i].displaySelectLabelOffsetSize: Number

选中的点，展示对应label的向上方向的偏移距离

# series[i].stackName: String

堆积序列，目前支持堆积序列的只有以下类型

	- 'bar'
	- 'line'

# series[i].displayLabels: Boolean

是否在图表上展示对应节点的数值标签

# series[i].name: String

数据序列的名称

# series[i].xAxisIndex: Number

数据序列所依赖的x轴索引号，默认为0

[注] 目前所支持最大坐标轴个数为2，索引号>=2时当前数据不会显示。

# series[i].yAxisIndex: Number

数据序列所依赖的y轴索引号，默认为0。

[注] 目前所支持最大坐标轴个数为2，索引号>=2时当前数据不会显示。

# series[i].lineWidth: Number

数据序列控制的线条粗细，仅对以下类型生效

	- 'line'
	- 'stackline'
	- 'radar'
	- 'rose'(控制边缘粗细)
	- 'pie'(控制边缘粗细)

# series[i].lineColor: Color

当图形为饼图时，设置饼图边缘线的颜色

# series[i].isDashLine: Boolean

数据序列控制的线条样式，如果设置为true显示虚线，反之显示实线。仅对以下类型生效

	- 'line'
	- 'stackline'

# series[i].isLineFillEnabled: Boolean

数据序列控制的线条填充样式，如果设置为true则添加填充，反之只显示线条。仅对以下类型生效

	- 'line'
	- 'stackline'
	- 'radar'

# series[i].lineFillColor: Color

数据序列控制的线条的填充颜色，如果不设置则和color属性取相同的值。仅对以下类型生效

	- 'line'
	- 'stackline'
	- 'radar'

# series[i].isCubicEnabled: Boolean

数据序列控制的线条的线条样式，如果设置为true则曲线显示，反之只显示线条。仅对以下类型生效

	- 'line'

# series[i].barGroupGapSize: Number

柱状图组间的空隙大小比例。barGroupGapSize * 单位距离为实际空隙。barGroupGapSize ∈ [0, 1]

# series[i].barGapSize: Number

柱状图组内柱间的空隙大小比例。barGapSize * 单位距离为实际空隙。barGapSize ∈ [0, 1]

# series[i].barRadiusSize: Number

柱状图的圆角，仅在末端生效，该值会在超出柱状宽度时被裁剪为可以设置的最大宽度。

# series[i].color: Color

数据的颜色，当为柱状图或者堆积柱状图时控制柱子的颜色。
当为线图或者堆积线图时候控制线条的颜色。
当为饼图或者玫瑰图时候控制边缘线的颜色。

# series[i].colors: Array[Color]

当为饼图、环图时，这个配置设置数据对应每个部分所对应的颜色

# series[i].displayJoint: Boolean

是否显示链接点。

# series[i].displayLegend: Boolean

是否展示图例，用于单一控制图例的展示。

# series[i].displayStartCircle: Boolean

是否展示小圆球 仅对饼图生效

# series[i].displayArchors: Boolean

是否展示线上的锚点，设计上只对line生效。

# series[i].archorSize: Number

线上锚点的大小。这些锚点外部的颜色一定是跟随线的颜色。

# series[i].archorInnerSize: Number

线上锚点内部小圆球的大小。

# series[i].archorInnerColor: Color

线上锚点内部小圆球的颜色

# series[i].innerRatio: Number

饼图内环占外环的比

# series[i].outterRatio: Number

饼图外环占画布的比

# series[i].indicators: Map

用于配置饼图周围的连线和值的展示
# series[i].indicators.style: String
只能取以下值
- 'point-to' 设置这个值作指向文字的样式
- 'around' 设置这个值作被文字上下包围的样式。读取data[j].text作为线上端的文字，data[j].desc作为低端文字
# series[i].indicators.displayStartCircle: Boolean
当设置为true时在连线起始点绘制一个和线条颜色相同的小圆球
# series[i].indicators.data: Array
用于配置展示文字
# series[i].indicators.data[j].text: String
用于表示指向或者包围时候的文字
# series[i].indicators.data[j].desc: String
用于表示包围文字底部内容
# series[i].indicators.data[j].lineColor: Color
用于表示包围文字底部内容
# series[i].indicators.data[j].lineWidth: String
用于表示包围文字底部内容
# series[i].indicators.data[j].textColor: Color
文字颜色
# series[i].indicators.data[j].textFontSize: Font
文字样式
# series[i].indicators.data[j].descColor: Color
底部文字颜色
# series[i].indicators.data[j].descFontSize: Font
底部文字样式

# API

# 选择事件

响应点击事件、长按等事件。

onHighlightChanged: function(selection, gestureType)

gestureType的值可能为：

  • tap
  • double-tap
  • pan
  • pinch
  • long-press

# 柱图、线图、饼图

选取的回调参数

  • selection x轴选择的值。
  • gestureType 当前手势类型，值为'pan'和'tap' 分别代表滑动手势和点击手势

# 散点图

  • selection 散点图返回x,y选择的值
  • gestureType 当前手势类型，值为'pan'和'tap' 分别代表滑动手势和点击手势

# 方法

设定选中位置的方法：

focus: function(xIndex, serieIndex)
  • xIndex 在该组数据($series[serieIndex]$)中的第几个数据, 如果serieIndex没有传，默认选中第一组数据
  • serieIndex 用于索引数组

# 一些约定

# Color

  • 纯色：可以填写 #ffffff 或者 #fff 的字符串表示颜色，也可以通过 #ffffffaa八位或者#fffa的方式设置带alpha的纯色。
  • 渐变色： 目前只支持线性渐变，其格式如下：
{
  type: 'lineargradient',
  start: [0, 1],
  end: [0, 0],
  components: [
    { color: '#f6e58d', offset: 0 },
    { color: '#ffbe76', offset: 0.2 },
    { color: '#ff7979', offset: 1 },
  ]
}

# 一些约束

# 饼图

若希望展示饼图或者玫瑰图，则设置series属性的第一条数据的type值为`pie`或者`rose`.并且其他的值会被忽略，仅仅展示一条。
若希望展示雷达图，需要保证设置series属性的第一条数据的type值为'radar'，注意其他的值不会被忽略。

# 混合图表

目前安卓端不能将气泡图与柱图、线图、堆积图等进行混合。

# 坐标轴

目前横纵坐标轴各最大支持2条轴，依赖超过这个限制的数据不会展示。

# 堆积图

堆积线图与堆积柱图目前必须分别设置type为`stackline`与`stackbar`，仅仅在type为`line`或者`bar`上设置stackName不生效。
