# AudioRecorder

音频录制。

# 方法

# prepare(options, callback)

录音准备。需要传入一些配置项。

# 参数

sampleRate {Number} 采样率，默认值：44100；
channels {Number} 声道，可选值：1（单声道）、 2(立体声)；
encoding {String} 编码：
android支持: amr_wb/aac/he_aac(api>16)/aac_eld(api>16)/vorbis(api>21)/amr_nb
iOS支持: lpcm/ima4/aac/MAC3/MAC6/ulaw/alaw/mp1/mp2/alac/amr；
quality {String} 音频质量，可选值：low、medium、high。(仅支持iOS)；
outputFormat {String} 输出格式。(仅支持android)：
android支持: mpeg_4/aac_adts(api>16)/amr_nb/amr_wb/webm/3gp；
callback {Function} function(ret){} 用户操作完成后的回调，回调函数的参数 ret 的格式如下：
ret.data.uri {String} 文件地址，可使用audio模块播放；
ret.data.time {Number} 录音时长，单位：ms；
ret.data.meter {String} 音量大小，(仅支持iOS)；
ret.data.error {String} 错误详情。

注: 参数均有默认值，可传入空对象。

# start()

开始录音。

# stop()

停止录音。

# 示例

<template>
	<div style="width:750;">
	  <div style="width:750;height:100;background-color:#00ff00;justify-content:center;align-items:center">
	    <text style="font-size:30;">audio-recorder module test</text>
	  </div>
	  
	   <div onclick="prepare" style="width:750;height:100;margin-top:50;background-color:#00ff00;justify-content:center;align-items:center">
	    <text style="font-size:30;">prepare</text>
	  </div>
	  
	   <div onclick="start"  style="width:750;height:100;margin-top:10;background-color:#00ff00;justify-content:center;align-items:center">
	    <text style="font-size:30;">start</text>
	  </div>
	  
	   <div onclick="stop" style="width:750;height:100;margin-top:10;background-color:#00ff00;justify-content:center;align-items:center">
	    <text style="font-size:30;">stop</text>
	  </div>
	  
    <div onclick="play" style="width:750;height:100;margin-top:50;background-color:#0ff0a0;justify-content:center;align-items:center">
	    <text style="font-size:30;">play</text>
	  </div>

    <div style="width:750;height:400;margin-top:10">
      <text style="font-size:30;">{{result}} || {{url}}</text>
    </div>
	  

	</div>
</template>


<script>
  var audio = require('@weex-module/audio');
  var recorder = require('@weex-module/audio-recorder');
	module.exports = {
		data : {
	    result:'no result',
	    url:'nothing'
		},
		methods : {
	    prepare : function() {
	      var self = this;
	      recorder.prepare({},function(e){
	        self.result = JSON.stringify(e)
	        if(e.status === 'ended') {
	          self.url = e.data.uri;
	        }
	      })
	    },
	    start : function(){
	      recorder.start();
	    },
	    stop : function(){
	      recorder.stop();
	    },
	    play:function(){
	      var self = this;
        audio.load({
          id:1,
          url:self.url,
          loop:true,
          autoplay:true
        })
	    }
	    
		},
		ready : function() {
		  
		},
	}
</script>
