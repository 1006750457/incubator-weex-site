# CodeGenerator

xcode-generator用于基于设定内容生成对应二维码或条形码图片。

# 方法

# generate（params, callback）

# 接口

参数

  • params{map}:内容及参数，类型为字典。可传入值有：
    • format :生成的码格式，支持格式包括：
      • qr_code: 二维码
      • code_128 : CODE_128格式的条形码
    • code : 编码内容
    • width: 生成的码图片宽度（最小50px）
    • height :生成的码图片高度（最小50px）
    • bgColor :生成的码图片背景色（默认白色）
  • callback {Function} function (ret){}：生成结果回调：
    • ret.result {String}: 结果类型，success 、error；
    • ret.data {String}: 结果内容，成功时为本地临时文件路径，失败时为错误原因； 注：
  • params 内 format 为 qr_code时，生成的图片尺寸为正方形，取width作为边长，height值无效;

# 使用示例

<template>
  <scroller class="wrapper">
    <div class="input-cnt">
      <text class="input-label">URL:</text>
        <input type="text" class="input" @input="oninput"/>
    </div>
    <div class="btn" @click="open">
      <text class="btn-txt">生成二维码</text>
    </div>
    <image class="code-image" :src="codeUrl"></image>
  </scroller>
</template>

<script>

  const modal = weex.requireModule('modal');
  const CodeGenerator = weex.requireModule('xcode-generator');

  export default {
    data: () => ({
      url: '',
      codeUrl: ''
    }),
    methods: {
      oninput (event) {
        this.url = event.value;
      },
      open () {
        if(this.url === '') {
          this.codeUrl = '';
          modal.toast({
            message: '请输入要生成的二维码内容！',
            duration: 0.3
          })
        } else {
          if(CodeGenerator) {
            CodeGenerator.generate({
              format: 'qr_code',
              code: this.url,
              width: 512,
              height: 512
            }, (ret) => {
              if (ret.result === 'success') {
                this.codeUrl = ret.data;
                if (ret.result.indexOf('file://') === -1) {
                  this.codeUrl = `file://${this.codeUrl}`;
                }
              } else {
                modal.toast({
                  message: '十分抱歉，二维码生成失败！',
                  duration: 0.3
                })
              }
            });
          }
        }
      }
    }
  }
</script>
