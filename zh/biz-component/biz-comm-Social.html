# Social

xsocial是基于友盟的社会化分享、授权模块。

# 方法

# share（platform，extras, callback）

# 分享接口

参数

  • platform{string}：分享平台，可选值有"weixin", "weixin_timeline", "qq", "qzone", "weibo"
  • extras{map}:分享的内容，类型为字典。可传入值有：
    • title :分享的title
    • content :分享的文字内容
    • img:分享的图片路径
    • url :分享的链接
  • callback {Function} function (ret){}：分享回调：
    • ret.wx_status:状态，有start、complete、error、cancel
    • ret.wx_errCode:错误码
    • ret.wx_platform:平台
    • ret.wx_errMsg：错误原因 注：分享分两种情况
  • extras 内 url 不为空，则以网页链接的分享的形式分享（ title 为分享网页名， content 为分享的简要内容， img 为缩略图片）
  • extras 内 url 为空，图片分享形式，只分享图片， 取img 字段

# showShareBoard(platforms, callback)

打开分享面板 参数：

  • platforms{array}：分享平台
  • callback {Function} function (ret){}：分享回调：
    • ret.wx_platform：分享平台

# authorize(platform, callback)

# 授权接口

参数:

  • platform{string}:平台
  • callback {Function} function (ret){}：授权回调
    • ret.wx_status:状态 start、cancel、complete、error
    • ret.wx_errCode:错误码
    • ret.wx_errMsg:错误信息
    • ret.wx_platform:平台
    • 其他key：授权成功情况下的相关参数，具体可以参考：友盟官网

# 初始化

# iOS

如果你引入了 EmasSocial 组件，因为其底层引入的是友盟分享套件，您可以移步友盟官网https://dev.umeng.com/sdk_integate/ios-integrate-guide/sharehttps://dev.umeng.com/sdk_integate/ios-integrate-guide/share#1https://dev.umeng.com/sdk_integate/ios-integrate-guide/share(https://dev.umeng.com/sdk_integate/ios-integrate-guide/share#1) 查看初始化流程，在 URL Types和 plist配置完成以后，也可以使用我们的SDK做初始化，初始化代码示例如下：

#import <EmasSocial/XCOSocial.h>

// 初始化友盟
[[XCOSocial sharedInstance] initUMWithUMAppKey:@"um_app_key" appSecrete:nil];

// 初始化 QQ
[[XCOSocial sharedInstance] setPlaform:UMSocialPlatformType_QQ appKey:@"app_key" appSecret:@"app_secret" redirectURL:@"http://mobile.umeng.com/social"];

// 初始化 Weibo
[[XCOSocial sharedInstance] setPlaform:UMSocialPlatformType_Sina appKey:@"app_key" appSecret:@"app_secret" redirectURL:@"http://sns.whalecloud.com/sina2/callback"];

// 初始化 微信
[[XCOSocial sharedInstance] setPlaform:UMSocialPlatformType_WechatSession appKey:@"app_key" appSecret:@"app_secret" redirectURL:@"http://mobile.umeng.com/social"];

# Android

Android端若要使用该模块，需要在初始化时完成相关注册:

SocialManager.getInstance().registerShareService(UmengShareServiceImpl.class);
SocialManager.getInstance().registerAuthService(UmengAuthServiceImpl.class);
Map<String, String> extras = new HashMap<>();
extras.put("appkey", "");
extras.put("secret", Constants.UMENG_SECRET);
extras.put("channel", Constants.UMENG_CHANNEL);
SocialManager.getInstance().getShareService().init(this, extras);

SocialManager.getInstance().getShareService().setQQ(Constants.QQ_APPID, Constants.QQ_APPKEY);
SocialManager.getInstance().getShareService().setWeibo(Constants.WEIBO_APPKEY, Constants.WEIBO_APPSECRET, Constants.WEIBO_REDIRECT_UTL);
SocialManager.getInstance().getShareService().setWeixin(Constants.WEIXIN_APPID, Constants.WEIXIN_APPSECRET);

# 使用示例

<template>
    <div>
        <text class="text" @click="share('weibo', 'url')">
            weibo 链接分享
        </text>
        <text class="text" @click="share('weibo', 'pic')">
            weibo 图片分享
        </text>
        <text class="text" @click="auth('weibo')">
            weibo 授权
        </text>
        <text class="text" @click="share('qzone', 'url')">
            qq 链接分享
        </text>
        <text class="text" @click="share('qzone', 'pic')">
            qq 图片分享
        </text>
        <text class="text" @click="auth('qq')">
            qq 授权
        </text>
        <text class="text" @click="share('weixin', 'url')">
            weixin 链接分享
        </text>
        <text class="text" @click="share('weixin', 'pic')">
            weixin 图片分享
        </text>
        <text class="text" @click="auth('weixin')">
            weixin 授权
        </text>
        <text class="text" @click="showBoard">
            打开分享面板
        </text>
    </div>
</template>

<script>
const modal = weex.requireModule('modal')
const social = weex.requireModule('xsocial')
export default {
    data() {
      return {
        imgurl:"https://vuejs.org/images/logo.png"
      }
    },
    methods: {
        share: function(platform, content) {
          var extras = {
            img:"https://vuejs.org/images/logo.png"
          }
          if (content === 'url') 
          {
            extras['title'] = 'title'
            extras['url'] = 'https://weex.apache.org/cn/references/components/image.html'
            extras['content'] = 'content'
          }
            
          social.share(platform, extras, function(e){
            modal.toast({
                message: JSON.stringify(e)
            })
          })
        },
        showBoard: function() {
          var list = ['qq', 'weixin', 'weibo']
          social.showShareBoard(list, function(e) {
            modal.toast({
              message: JSON.stringify(e)
            })
          })
        },
        auth: function(platform) {
          social.authorize(platform, function(e) {
            modal.toast({
              message: JSON.stringify(e)
            })
          })
        }
    }
}

</script>
