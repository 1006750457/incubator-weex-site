#<mask>

<mask> 组件用于全屏遮罩，默认全屏，加载到所有页面的最上层，可以添加任何子组件，提供如下功能：

  1. 提供遮罩层组件mask，覆盖状态栏和导航栏上，Android、iOS全遮罩；H5 遮罩当页，不包括浏览器bar
  2. 样式保持通用规则
  3. 高宽可以设置，通知栏可以通过mask实现

# 属性

参数描述类型默认值
width屏幕宽度Number
height屏幕高度Number
top到顶部屏幕距离Number0
position定位Stringandroid/ios 默认是absolute h5默认是fixed

目前native没有点击事件冒泡机制，假如用户不需要触发背景隐藏事件，需要添加一层div，截获当前事件。
Android端mask组件会被back键关闭，可以通过监听visiblechanged事件在mask显示或关闭时收到通知（仅android有效）

# 示例

<template>
  <div class="wrapper" >
    <image src="{{logoUrl}}" class="logo" onclick="toggle"></image>
    <mask class="mask" if="{{shown}}"  onclick="hide" style="background-color: rgba(0,0,0,0.5);">
      <div class="custorm" onclick="divclick">
        <text class="text" >遮罩层</text>
      </div>
    </mask>
  </div>
</template>
<style>
  .mask{
    background-color: rgba(255,255,0,0.3);
  }
  .wrapper {align-items: center; margin-top: 10px;}
  .title {font-size: 48px;}
  .custorm {background-color: red;font-size: 48px;margin-top: 100px;margin-left: 100px;height: 700px}
  .logo {width: 360px; height: 82px; }
</style>
<script>
  module.exports = {
    data: {
      logoUrl: 'http://alibaba.github.io/weex/img/weex_logo_blue@3x.png',
      target: 'World',
      shown:false,
      screenHeight: 0
    },
    ready: function () {
      this.screenHeight = this.$getConfig().env.deviceHeight ;
    },
    methods: {
      update: function (e) {
        this.target = 'Weex'
      },
      toggle: function () {
        this.shown = true
      },
      hide: function () {
        this.shown = false
      },
      divclick: function (e) {  //必须加这个，由于native没有加事件冒泡
      },
      toast: function (e) {
        var message = "test";
        var duration = 3;
        var params = {
          'message': message,
          'duration': duration,
        }
        this.$call('modal','toast',params);
      },
    }
  }
</script>
